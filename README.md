# 아침 뉴스 브리핑 📰

조선일보, 중앙일보, 동아일보의 주요 뉴스를 한눈에!

## 🚀 Vercel에 배포하기 (가장 쉬운 방법)

### 1단계: GitHub에 업로드

1. [GitHub](https://github.com)에 로그인
2. 오른쪽 상단 `+` 버튼 → `New repository` 클릭
3. Repository 이름: `news-briefing` (또는 원하는 이름)
4. `Public` 선택
5. `Create repository` 클릭
6. 생성된 페이지에서 "uploading an existing file" 링크 클릭
7. 다음 파일들을 드래그 앤 드롭:
   - `index.html`
   - `package.json`
   - `api/rss.js` (api 폴더 채로)

### 2단계: Vercel에 배포

1. [Vercel](https://vercel.com) 접속
2. `Sign Up` → GitHub 계정으로 로그인
3. `Add New...` → `Project` 클릭
4. 방금 만든 GitHub 리포지토리 선택
5. `Deploy` 버튼 클릭!

🎉 **완료!** 1-2분 후 자동으로 배포됩니다.

### 3단계: 접속하기

- Vercel이 자동으로 URL을 생성해줍니다 (예: `https://news-briefing-xxx.vercel.app`)
- 이 URL을 북마크에 추가하세요!
- 매일 아침 접속하면 최신 뉴스를 볼 수 있습니다

## 📱 기능

- ✅ **실시간 뉴스**: RSS 피드로 항상 최신 뉴스
- ✅ **카테고리별 분류**: 전체, 정치, 사회, 경제, 과학/기술
- ✅ **3대 언론사**: 조선, 중앙, 동아
- ✅ **모바일 최적화**: 스마트폰에서도 완벽
- ✅ **완전 무료**: 비용 없음!

## 💡 팁

1. **스마트폰 홈화면에 추가**
   - Safari (iPhone): 공유 버튼 → 홈 화면에 추가
   - Chrome (Android): 메뉴 → 홈 화면에 추가

2. **매일 아침 루틴**
   - 아침에 일어나서 이 앱을 먼저 열기
   - 5분 안에 주요 뉴스 파악!

3. **커스텀 도메인 (선택사항)**
   - Vercel에서 무료로 커스텀 도메인 연결 가능
   - 예: `news.yourname.com`

## 🔧 문제 해결

### 뉴스가 안 보여요
- 잠시 기다려주세요 (RSS 로딩 중)
- 새로고침 버튼 클릭
- 브라우저 캐시 삭제 후 재시도

### 배포가 안 돼요
- GitHub 리포지토리가 Public인지 확인
- api 폴더가 제대로 업로드됐는지 확인
- Vercel에서 에러 로그 확인
